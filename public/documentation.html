API Documentation

GET /api/filter/hello

Returns a welcome message.

Example Response:

{"message": "Hello! Welcome to Products Catalog!!!"}

GET /api/products

Fetches all available products in the database.

Example Response: JSON array of products.

GET /api/products/filter

Fetches products with optional filters.

Query Parameters:

  • id - Filter by product ID
  • name - Filter by product name (case insensitive)
  • numericFilters - Apply numeric filters on fields such as price or rating. Use operators like >, >=, =, <, and <=.

Example Query: /api/products/filter?numericFilters=price>100,rating>=4
